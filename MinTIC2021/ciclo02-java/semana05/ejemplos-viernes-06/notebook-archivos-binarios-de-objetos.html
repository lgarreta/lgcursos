Clases para archivos binarios serializable¶

  • Se escriben en y leen en formato binario objetos completos:

  • Clases para la Lectura:

    • FileInputStream : Dispositivos de lectura desde archivos
    • ObjectInputStream : Flujo de datos binario para leer

  • Clases para la Escritura:

    • FileOutputStream : Dispositivos de escritura en archivos
    • ObjectOutputStream : Flujo de datos binario para enviar

  • Los clases de objetos que se van a escribir/leer deben ser serializable, es decir implementar la interface Serialiable

Escritura/Lectura en binario de Objetos de Clase Estudiante¶

In [11]:
public class Estudiante implements Serializable {
    // Atributos
    String nombres;
    String correoInstitucional;
    long numeroCelular;
        
    // Constructor
    public Estudiante () {
        this.nombres = "";
        this.correoInstitucional = "";
        this.numeroCelular = 0;
    }
    
    public Estudiante (String nombres, String correoInst, long nroCelular){
        this.nombres = nombres;
        this.correoInstitucional = correoInst;
        this.numeroCelular = nroCelular;
    }
    
    // Metodos
    public void presentarse () {
        System.out.println ("NOMBRES: " + nombres + " CORREO: " + correoInstitucional + " CELULAR" + numeroCelular);
    }            
}

Implementación serializable en el ejemplo de estudiantes¶

  • Solo se modifica el DAO
    • Se crea un metodo guardarse() que se ejecuta cada que se modifica la lista del DAO
      • El metodo guardarse() escribe siempre toda la lista desde el inicio al final, es decir, el archivo siempre está actualizado.
      • Se guarda primero el numero de elementos y después se recorre la lista guardando cada estudiante
    • Se crea un método recuperarse que se ejecuta cuando en el metodo crear del DAO, es decir, cuando el controlador crea el DAO para iniciar el programa.
      • El metodo recuperarse abre el archivo y recupera primero el numero de elementos y despues en un ciclo recupera estudiante por estudiante y lo agrega a la lista del DAO.
